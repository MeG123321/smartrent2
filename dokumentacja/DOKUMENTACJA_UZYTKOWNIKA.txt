## 1. Introduction
Welcome to the user documentation for the smartrent2 application, designed to simplify rental property management.

## 2. Login Screen
### Description
The login screen allows users to access their accounts.

### User Actions
- Enter email and password.
- Click 'Login'.

### Possible Messages
- "Login successful!"
- "Invalid credentials. Please try again."

### ASCII Diagram
```
[   Login Screen   ]
--------------------
| Email: ___________|
| Password: ________|
|                    |
|       [Login]     |
--------------------
```

## 3. Registration Screen
### Description
Users can create a new account using the registration screen.

### User Actions
- Fill out the registration form.
- Click 'Register'.

### Possible Messages
- "Registration successful!"
- "Email already in use."

### ASCII Diagram
```
[ Registration Screen ]
---------------------
| Name: _____________|
| Email: ____________|
| Password: _________|
|                    |
|     [Register]     |
---------------------
```
