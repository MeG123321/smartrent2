## Processes Overview
SmartRent2 is designed to facilitate the management of rental properties efficiently.

## Screen Descriptions
- **Login Screen**: Allows users to access their account.
- **Registration Screen**: Used for new user accounts.
- **Main Screen**: Home page after logging in.
- **User Panel**: Dashboard for user-specific actions.
- **Property Management**: Where users can manage rental properties.
- **Messaging System**: Interface for communication.
- **Rental History**: Section showing past rental details.
- **Support Tickets**: Feature for submitting queries.
- **Admin Panel**: For administrative tasks and overviews.

## User Instructions
### Login
1. Open the application.
2. Enter your username and password.
3. Click on the 'Login' button.
   
   ```
   +----------------+
   | Username       |
   | [____________] |
   +----------------+
   | Password       |
   | [____________] |
   +----------------+
   |    [Login]     |
   +----------------+
   ```

### Registration
1. Click on the 'Register' link.
2. Fill in the required details (username, email, password).
3. Submit the form.

### Main Screen
- Shows an overview of your rental properties and notifications.

### User Panel
- Access personal settings and statistics.

### Property Management
- To add a property:  
   1. Navigate to Property Management.  
   2. Click 'Add Property'.  
   3. Fill in the property details.  
   4. Save.
   ```
   +------------------+
   |   Add Property    |
   +------------------+
   | Property Name    |
   | [______________] |
   | Price            |
   | [______________] |
   +------------------+
   |     [Save]      |
   +------------------+
   ```

### Messaging System
1. Go to Messaging.
2. Select a conversation or start a new one.
3. Type your message and hit send.

### Rental History
- View all past rentals and their details in this section.

### Support Tickets
1. Click on Support Tickets.
2. Fill in the issue description.
3. Submit the ticket.

### Admin Panel
- For admins, providing a comprehensive overview and control of the application.

> **Note**: All instructions should be followed carefully to ensure seamless operation of SmartRent2.

## Conclusion
This documentation serves as a comprehensive guide for users of SmartRent2, ensuring that each feature and functionality is properly understood and utilized.
